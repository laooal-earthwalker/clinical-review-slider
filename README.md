## Overview
This repository contains a custom CSS implementation for a professional, clinical-grade Google Review slider. Designed specifically for **Boulder Pain Relief Massage, LLC**, it helps bridge the gap between clinical excellence and modern web performance.

## Key Features
- **HIPAA-Focused:** Only handles public-facing review data.
- **Clean Aesthetic:** Light-weight, medical-grade styling for health clinics.
- **Squarespace Ready:** Built to be easily injected into Squarespace Website Tools.

## How to Use
1. Copy the code from `styles.css`.
2. Paste it into your website's custom CSS editor.
3. Apply the class `.review-slider-container` to your testimonial elements.
